# V7 Phase 41: Service-aware policy foundation

Date: 2026-05-07

## Constraint

All work stays within the current single Russian V7 VPS:

```text
195.2.79.116
```

No second VPS, no mobile/residential/home egress, no additional external
trusted server.

## Goal

Start moving V7 from:

```text
user -> egress
```

to:

```text
user + destination/service -> route_class -> route behavior
```

This phase is intentionally a foundation layer. It does not change live user
routing or install new fwmark/table rules beyond the already existing direct
RU mechanism.
